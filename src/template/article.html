MAC安装 4.0.3

下载安装包 ..tgz

解压

重命名为mongodb，移动到 /usr/local

bin/目录加入环境变量 编辑~/.zhsrc ? ~/.bash_profile

export PATH=/usr/local/mongodb/bin:${PATH}

使配置问价生效

source ~/.bash_profile

根目录下创建数据目录 最高权限

sudo mkdir -p /data/db

打开mongod提示/data/db只有可读权限 增加权限

sudo chown id -u /data/db

填写用户名 给予权限

sudo chown ZHANGJIANHUA /data/db

配置

无论WIN，MAC都要自己建立目录，和配置信息。

新增目录结构：

├── conf
│   └── mongod.conf
├── data
├── log
│   └── mongod.log
基本数据类型 对应包装类 默认值
byte Byte 0
short Short 0
int Integer 0
long Long 0L
float Float 0.0f
double Double 0.0d
char Charactor '\u0000' (空格)
boolean Boolean false 
public void test(int i) {
Integer var1 = 20 + i;
if(var1 % 5 == 0)
  System.out.println('能被5整除');
else
  System.out.println('不能被5整除');
}

mongod.conf配置信息：

port = 12345
dbpath = data
logpath = log/mongod.log
fork = true // 以后台程序运行

启动

找到安装目录 bin的上一级

MAC: /usr/local/mongodb/


```./bin/mongo`

以配置文件启动

./bin/mongod -f conf/mongod.conf

连接mongodb服务器 / 数据库

查看连接具体参数

./bin/mongo --help

链接数据库：

./bin/mongo 127.0.0.1:12345/foo

关闭mongodb服务器

在(./mongo)已经开启时，关闭：

use admin

db.shutdownServer()

基本操作

查看数据库列表

show dbs

选择数据库 / 如果没有则创建此数据库并进入

use <db_name>

选择数据库后 删除数据库 删除后进行检查 show dbs

db.dropDatabase()

集合(表)。 插入 文档(记录) / 若没有 默认创建此集合。检查 show dbs

db.<table_name>.insert(<json>)

// 用js语法批量插入多条数据
for(i = 3; i < 100; i++)db.<table_name>.insert({x: i})

查看集合(表)列表

show collections

查询 文档(记录)

可以不带参数 默认显示所有 每条文档生成全局唯一_id 此_id也可以自定义 但不可以重复

db.<table_name>.find()

带参

db.<table_name>.find(<json>)

文档多条时 计数(length)查询

db.<table_name>.find().count() //99

查询[3 - 50]文档 查询第3条开始 返回2条文档 同时以x排序

db.<table_name>.find().skip(3).limit(2).sort({x: 1})
